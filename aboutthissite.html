1. General Description of This Site

This website which I have constructed is an official website for an Animation company, 'Visionary'. The site consists of 5 working and interactive pages: Home (index.html), Products (products.html), About Us (aboutus.html), Login (login.html) and About This Site (aboutthissite.html). The webpages also work without internet connection as there are no external links to 'https' websites.

 

The web page structures mainly consist of: The header/navigation bar, the body section(s) and the footer. There is also a favicon displayed in every page. The navigation bar clearly shows the webpages that this site is consisted of.

 

The features included in the navigation bar are listed below:

  • A logo
  • Flex box functions
  • Hyperlinks to webpages
  • Interactive features such as hover functions
  • Dropdown box (in the 'Register' button)

    •  

    The body sections of my site includes these features:
  • Flex box functions
  • Hyperlinks to other webpages
  • Formatted headings and text
  • Well displayed and formatted images and galleries
  • Interactive features such as hover functions and text decorations (borders, underline etc.)
  • Special features (input and javascript in the 'Log In' page)

    •  

    The footer sections of my site includes these features:
  • A logo
  • Flex box functions
  • Formatted headers and text
  • Column functions
